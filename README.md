# Sistema de Gestión Psicológica

Proyecto académico desarrollado con ASP.NET MVC.

## Descripción
Sistema web orientado a la gestión de citas psicológicas entre pacientes y psicólogos.

## Tecnologías
- ASP.NET MVC (C#)
- SQL Server
- Entity Framework
- HTML
- CSS
- Bootstrap
- IIS
